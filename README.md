# Moody Tips

Edit `moody_tips.json` to change the tips shown in the Moody app. The app picks up changes on its own within a short while, no app update needed.

Each stage (calm, horny, sensitive, danger) has a `_yes` list (things to do) and a `_no` list (things to avoid). Just change the text inside the quotes. Keep the structure exactly as is, and make sure every line in a list except the last one ends with a comma.

The file must stay valid JSON. If an edit breaks it, the app simply keeps showing the last working tips, so nothing breaks on the users' side.
